Overfloe

A modern desktop IRC client for a more civilized hacker.

Floe started as a very wonderful minimalist IRC client in .NET / WPF for Windows. The original project has been dead for years, with some others making little fixes and branching around. Overfloe is a fork of @dezro's branch of Floe, and incorporates fixes and updates for the UI.

Want to pitch in?

We use VS 2013 Community for Floe development. If you want to help out, fork our Floe repository, and send us a PR!
